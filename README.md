# Weather web app v2:
this is a second shot at the weather app improving on the last one in every aspect. 

## Changes over the previeous one: 
- Using Typescript instead of Javascript
- Integrating Redux toolkit for state management
- Replacing WeatherAPI with [Open-meteo](https://open-meteo.com), [Geonames](https://www.geonames.org/), [Open street map](https://openstreetmap.maps.arcgis.com/home/index.html) Api's and [Leaflet React](https://react-leaflet.js.org/) libarary
- Major design and performance improvements
